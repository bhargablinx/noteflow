# Personal Logs

There will be two layouts which will have different visuals:

## Layout 1

- This will be the home section.
- Whenever someone visits they will be land in this page.
- The sidebar will contain list of all the notes that have been created.
- The main section will show card where the notes will be there in card format.
- These note cards can be clicked by users and it will open **Layout 2**.

## Layout 2

- When the user clicks it it will open the notes
- And there will be two sections:
    1. Edit the notes (raw markdown will be shown)
    2. Preview notes
